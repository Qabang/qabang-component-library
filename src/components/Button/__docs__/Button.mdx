# Button

Button component with different props.

#### Example

<Canvas of={Button.TextAndIconLeft} />

## Usage

```ts
import {Button} from "qabang-component-library";

const Example = () => {
  return (
      <Button
        button_type=ButtonType.Primary
        text={"Button"}
        onClick={()=> console.log("Clicked")}
        disabled: false,
        icon: { icon: "image" },
        icon_position: "left",
      />
  );
};

export default Example;
```

#### Arguments

- **text** - _string_ - Represents the text content of the button.
- **button_type** - _ButtonType_ - Sets the type of the button e.g ButtonType.Primary
- **disabled** - _boolean_ - indicating whether the button should be disabled or not. When disabled, the button cannot be clicked or interacted with.
- **icon** - _MaterialSymbolProps_ - A MaterialSymbol icon.
- **icon_position** - _string "left | right"_ -  Positions the icon to the left or right side of the button text.
- **onClick** - _Callback_ - A function that is called when the button is clicked. It receives a MouseEventHandler for handling the click event on the button element.
